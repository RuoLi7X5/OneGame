# OneGame - 脑力训练站

OneGame 是一个专注于**脑力开发与训练**的 Web 小游戏集合平台。与常见的休闲娱乐不同，本站致力于提供高质量的益智挑战，涵盖逻辑推理、判断力、专注力、反应力与观察力等多个维度，旨在帮助用户在静谧的氛围中锻炼思维，提升核心认知能力。

虽然目前收录的游戏尚在起步阶段，但OneGame的目标是打造一个全面、专业的脑力训练基地。

在视觉呈现上，项目追求**中式素雅与写意留白**的美学风格。我们希望通过极简、轻雅的界面设计，为高强度的脑力训练创造一个宁静致远的思考空间，让“硬核”的训练内容与“柔美”的视觉体验达成和谐统一。

## 🎮 包含游戏

目前已上线两款基础训练项目：

### 1. 成语猜词 (Chengyu Caici)
一款基于拼音线索的成语逻辑推理游戏。
- **核心玩法**：玩家需根据反馈的声母、韵母、声调提示（绿色为正确，橙色为位置不对，灰色为不存在），在有限次数内猜出目标四字成语。
- **特色功能**：
  - **每日一题**：每天更新一道题目，支持本地记录连胜统计。
  - **练习模式**：随时开始的随机题目练习。
  - **释义展示**：通关后展示成语的详细释义与出处。

### 2. 舒尔特表格 (Schulte Grid)
一款经典的注意力与周边视力训练工具，融入了《小王子》主题元素。
- **核心玩法**：在 N×N 的网格中，按顺序尽可能快地点击从 1 到 N² 的数字。
- **高度定制**：
  - **多规格**：支持 3×3 至 8×8 多种网格尺寸。
  - **多模式**：包含“简单模式”（数字不动）与“常规模式”。
  - **特色玩法**：支持“奇数”、“偶数”等变体挑战。
- **视觉主题**：搭配小王子与狐狸的温馨插画，缓解训练疲劳。

## 🎨 设计风格

项目整体追求**新中式轻雅**美学：

- **视觉氛围**：主页背景采用动态落叶效果（红枫与青竹随机飘落），配合暖色调的径向渐变背景，营造秋日静谧氛围。
- **界面质感**：UI 组件采用**高透磨砂玻璃 (Frosted Glass)** 风格，卡片具有物理厚度感、光泽感与精致的边缘反光。
- **排版字体**：标题使用书法感强的宋体（思源宋体/方正清刻本悦宋），辅以“水墨晕染”文字效果，突显文化底蕴。

## 🛠️ 技术栈

本项目为纯静态 Web 应用 (Static Web App)：
- **HTML5**: 语义化标签结构。
- **CSS3**: 使用 Flexbox/Grid 布局，大量运用 `backdrop-filter`、`linear-gradient`、`box-shadow` 实现复杂的玻璃拟态效果与动效。
- **JavaScript (ES6+)**: 原生 JS 实现游戏逻辑，无重型框架依赖，轻量高效。
- **Canvas**: 首页落叶飘零动画实现。

## 🚀 如何运行

由于项目为纯静态文件，你可以通过任何静态文件服务器启动。

### 方法 1：使用 Node.js (推荐)
如果你安装了 Node.js，可以使用 `http-server` 快速启动：

```bash
# 安装 http-server (如果未安装)
npm install -g http-server

# 在项目根目录下运行
npx http-server -p 8080 -c-1
```
访问 `http://127.0.0.1:8080` 即可预览。

### 方法 2：使用 Python
```bash
# Python 3.x
python -m http.server 8080
```

### 方法 3：VS Code 插件
使用 "Live Server" 插件，右键 `index.html` 选择 "Open with Live Server"。
